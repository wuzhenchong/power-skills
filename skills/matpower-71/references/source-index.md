## Local MATPOWER Help Commands

Use these inside MATLAB/Octave when MATPOWER is installed:

```matlab
help caseformat
help runpf
help rundcpf
help runcpf
help runopf
help rundcopf
help mpoption
help add_userfcn
help run_userfcn
help opf
```

## Source Lookup Commands

```matlab
which runpf
which runopf
which opf_setup
which ext2int
which caseformat
```

When project-local overrides exist, `which` is the first check for whether MATLAB/Octave will use the intended file.
