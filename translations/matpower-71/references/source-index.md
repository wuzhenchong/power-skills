- MATPOWER GitHub 仓库：https://github.com/MATPOWER/matpower

- 所有 MATPOWER 手册：https://matpower.org/doc-original/manuals/
- MATPOWER 7.1 在线函数参考索引：https://matpower.org/docs/ref/matpower7.1/lib/menu7.1.html
- `caseformat` 7.1：https://matpower.org/docs/ref/matpower7.1/lib/caseformat.html
- `runpf` 7.1：https://matpower.org/docs/ref/matpower7.1/lib/runpf.html
- `runopf` 7.1：https://matpower.org/docs/ref/matpower7.1/lib/runopf.html
- Callback 阶段和例子：https://matpower.app/manual/matpower/CallbackStagesandExample.html

## 本地 help 命令

```matlab
help caseformat
help runpf
help rundcpf
help runcpf
help runopf
help rundcopf
help mpoption
help add_userfcn
help run_userfcn
help opf
```

## 源码定位命令

```matlab
which runpf
which runopf
which opf_setup
which ext2int
which caseformat
```

使用项目本地源码副本时，`which` 是确认 MATLAB/Octave 是否会调用预期文件的第一步。
